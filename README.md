# 我的小工具集合

这个仓库包含了一些实用的小工具和示例页面。

## 📁 项目结构

```
github_project/
├── python/
│   └── base64_converter.py    # Base64编码解码工具
├── news_site/
│   └── world_politics_news.html  # 世界政治新闻页面
└── README.md                   # 本文件
```

## 🛠️ 工具说明

### 1. Base64转换程序 (`python/base64_converter.py`)
一个简单的命令行工具，支持：
- 文本 ⇄ Base64 双向转换
- 交互式菜单操作
- 支持中文和特殊字符

**使用方法：**
```bash
cd python
python base64_converter.py
```

### 2. 世界政治新闻页面 (`news_site/world_politics_news.html`)
一个响应式的HTML页面，展示近期世界政治动态：
- 覆盖亚洲、欧洲、美洲、中东四个地区
- 卡片式设计，悬停效果
- 移动端适配
- 免责声明

**使用方法：**
直接在浏览器中打开 `world_politics_news.html` 文件即可。

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone <你的仓库地址>
cd github_project
```

2. 运行Base64转换器：
```bash
cd python
python base64_converter.py
```

3. 查看新闻页面：
   用浏览器打开 `news_site/world_politics_news.html`

## 📝 注意事项

- Base64转换程序需要Python 3.x环境
- 新闻页面为示例内容，不构成实际新闻报导
- 所有代码均为学习演示用途
